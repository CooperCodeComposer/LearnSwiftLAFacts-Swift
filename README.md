# LearnSwiftLAFacts-Swift
Demo app for AWS S3 image + json download in Swift 3

- Swift 3.0
- Xcode 8.0

##Features:
+ async downloading with URLSession
+ parsing Swift facts in json from AWS S3 
+ downloading an image from AWS S3
